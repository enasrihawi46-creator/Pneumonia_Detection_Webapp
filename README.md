## نظام كشف الالتهاب الرئوي (Pneumonia Detection)
هذا المشروع هو تطبيق ويب يستخدم الذكاء الاصطناعي لتحليل صور الأشعة السينية للصدر للكشف عن الإصابة بالالتهاب الرئوي.

## المميزات:
* **دقة عالية:** يعتمد على موديل VGG19 المتطور.
* **تحليل فوري:** يعطي النتيجة فور رفع الصورة.
* **تقارير طبية:** يتيح استخراج تقرير تشخيصي بصيغة PDF باللغة العربية.

##  التقنيات المستخدمة:
* **Python & Flask:** لبناء الموقع.
* **TensorFlow & Keras:** لتشغيل موديل التعلم العميق (Deep Learning).
* **ReportLab:** لإنشاء تقارير الـ PDF.

##  طريقة التشغيل:
1. قم بتثبيت المكتبات: `pip install -r requirements.txt`.
2. شغل ملف `app.py`.
3. افتح المتصفح على الرابط `http://127.0.0.1:5000`.
